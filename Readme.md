# Student Feedback Form - GitHub Push Exam Question

## Exam Question
**"Push project to GitHub remote repository. You have completed a student feedback form project locally. Push all local commits to the remote repository. Clone the repository into a different folder to verify synchronization."**

## Project Description
A complete Student Feedback Form web application built with HTML, CSS, and JavaScript. Features:
- Form validation for Name, Email, Course selection, and Feedback
- Responsive design with hover effects
- Client-side JavaScript for form submission handling
- No backend required - fully functional frontend

## Git Workflow Followed
1. Created project folder: `mkdir student-feedback-form`
2. Initialized Git: `git init`
3. Added project files: `git add .`
4. Committed changes: `git commit -m "Completed student feedback form project"`
5. Added remote: `git remote add origin https://github.com/EswarChowdary131013/Sem_Lab_Exam.git`
6. Pushed to main: `git push -u origin main`
7. Verified sync by cloning to new folder

## Files
- `index.html` - Main HTML structure
- `style.css` - Responsive styling
- `script.js` - Form validation and submission logic

## How to Run
Open `index.html` in any web browser. Fill form and submit to test functionality.
